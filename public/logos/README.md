# Company Logos

To add company logos locally:

1. Download the logos for each company:
   - Netflix: https://upload.wikimedia.org/wikipedia/commons/0/08/Netflix_2015_logo.svg
   - Amazon Prime: https://upload.wikimedia.org/wikipedia/commons/f/f1/Prime_Video.png
   - Disney+ Hotstar: https://upload.wikimedia.org/wikipedia/commons/6/64/Disney%2B_Hotstar_logo.svg
   - ZEE5: https://upload.wikimedia.org/wikipedia/commons/8/8a/ZEE5_logo.svg
   - SonyLIV: https://upload.wikimedia.org/wikipedia/commons/7/7a/SonyLIV_logo.svg
   - MX Player: https://upload.wikimedia.org/wikipedia/commons/7/7c/MX_Player_logo.svg
   - YouTube: https://upload.wikimedia.org/wikipedia/commons/b/b8/YouTube_logo_%282017%29.svg
   - Jio Cinema: https://upload.wikimedia.org/wikipedia/commons/9/9a/JioCinema_logo.svg

2. Save them in this directory with these filenames:
   - netflix.svg
   - prime-video.png (or .svg)
   - disney-hotstar.svg
   - zee5.svg
   - sonyliv.svg
   - mx-player.svg
   - youtube.svg
   - jio-cinema.svg

3. Update the component to use local paths first.
